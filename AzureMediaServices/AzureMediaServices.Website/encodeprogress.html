Encode Progress

This screen displays the Azure media encode job progress. The encoding job takes a considerable amount of time. The time taken depends on various factors. Like size of the video, duration of video, encoding job hardware configuration settings. Which is done in the azure portal. S3 configuration is the fastest. Also depends on number of tasks configured in azure portal.
